# @mosaicxr-ai/mcp-server

Node.js MCP (Model Context Protocol) server for Mosaic Bridge. Connects
MCP-compliant AI clients (Claude Desktop, Cursor, Claude Code, etc.) to a
Unity Editor running the Mosaic Bridge package.

**Status:** Pre-launch. 1.0 beta in preparation.

See the [repository root](https://github.com/MosaicXR-AI/mosaic-bridge) for
documentation, contribution guide, and license.

## Requirements

- Node.js 18 or later
- npm 9 or later
- Mosaic Bridge Unity package installed and running in a Unity Editor
